GetName
=======
The GetName function is used to retrieve the module name.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhab/Module/MRH_Module.h>


Syntax
------
.. code-block:: c

    std::string GetName() noexcept;


Parameters
----------
None.

Return Value
------------
The module name.
